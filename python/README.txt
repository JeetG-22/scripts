~~~~~~~~~~~~~~~~~ STEPS TO RUN SCRIPT ~~~~~~~~~~~~~~~~~

Download Python: https://www.python.org/downloads/

Create Python Virtual Environment:
1) In the directory where you place the script, requirments.txt and the excel files
type the command below in PowerShell to create the initial setup for the virtual
environment. Only need to do one time (dont include the arrows)
	---> python -m venv venv

2) To enter the virtual environemnt type
	---> .\venv\Scripts\Activate.ps1

3) To install the necessary libraries/modules type:
	---> pip install -r requirements.txt

4) run the .py file (note: make sure to change the paths in the script to
point to the excel files in your lcoal machine
	---> python VerizonNamelyScript.py
